Step-by-Step Customer Onboarding Guide (Website)
Step 1: Welcome Screen
Purpose: Introduce the customer to The Spice Rack Atlanta and set expectations.
Action: Display a brief welcome message (e.g., "Welcome to spice rack! Fresh tiffin delivered to your door or ready for pickup. Let’s get started!").
Features:
Option to "Sign Up" or "Log In" (for returning users).
Small banner showcasing a special (e.g., "First order 10% discount with code WELCOME").
Button to watch a 30-second intro video (optional, based on your preference).

Step 2: Account Creation
Purpose: Collect essential customer information securely.
Action: Prompt the customer to create an account.
Fields:
Full Name
Email Address (for order confirmations)
Phone Number (validated to ensure it’s not a Google Voice/office number; used for text reminders)
Password (with basic security requirements).
Features:
"Sign up with Google/Facebook" option for faster onboarding.
Checkbox: "I agree to receive text/email updates" (pre-checked but optional).
Link to Terms of Service and Privacy Policy for transparency.

Step 3: Address and Delivery Details
Purpose: Gather accurate delivery/pickup info to avoid order issues.
Action: Prompt for address details with clear labeling.
Fields:
Billing Address: Street address, city, state, ZIP code (mandatory for payment).
Shipping Address: Option to mark "Same as billing" or enter a separate address (street, apartment number, ZIP code).
Gate Code (if applicable).
Note Section: Free-text field for instructions (e.g., building name, "Leave with concierge," "Park in visitor lot").
Features:
ZIP code validation to confirm delivery area eligibility.
Dropdown for pickup location (if choosing pickup instead of delivery) with 6 options (toggleable by you in the back end).
Auto-suggest address feature using Google Maps API to reduce errors.

Step 4: Order Preference Setup
Purpose: Let customers choose how they want to order (one-time or subscription (future).
Action: Present two options with clear benefits.
Options:
One-Time Order: "Order whenever you want, starting at $30."
Subscription: "Save time with weekly tiffin, no hassle of remembering."
Subscription Details (if selected): (** FUTURE) 
Choose frequency: Weekly (Monday, Thursday, or both).
Select meals: Pre-set combo or custom items (e.g., 8oz Butter Chicken + Rice).
Confirm first delivery/pickup date (e.g., next Monday).
Features:
Highlight subscription perks (e.g., "Pause anytime, free dessert after 4 weeks").
Preview of current menu to entice immediate ordering.

Step 5: Payment Setup
Purpose: Secure payment method for orders and subscriptions.
Action: Prompt to add a payment method via Authorize.net integration.
Fields:
Card Number, Expiration Date, CVV, Name on Card.
Option to save card for future use (default checked).
Features:
Display $30 minimum order requirement (before taxes/delivery).
Option to enter a discount code (e.g., "WELCOME" for a free samosa).
Assurance message: "We won’t charge until your order is confirmed."

Step 6: Confirmation and Agreement
Purpose: Ensure customers understand their responsibilities and policies.
Action: Display a summary and require agreement.
Summary:
Order type (one-time/subscription), delivery/pickup details, payment method.
Agreements:
Checkbox: "I agree to pick up on time at the selected location" (for pickup orders).
Checkbox: "I understand delivery requires me to be available or provide access" (for delivery orders).
Link to "How It Works" page for more details.
Features:
Button: "Confirm and Start Ordering."

Step 7: First Order Placement
Purpose: Guide customers to place their first order immediately.
Action: Redirect to the menu page post-confirmation.
Features:
Display active menu (e.g., Monday’s menu if within order window).
Highlight upsell suggestions (e.g., "Add a samosa for $2").
Clear cart with $30 minimum reminder and option to apply discount codes.
Pop-up tip: "Order 48 hours in advance for best availability!"

Step 8: Welcome Confirmation
Purpose: Reassure customers and build trust.
Action: Send an immediate email/text after onboarding.
Content:
"Thanks for joining The Spice Rack Atlanta! Your account is ready."
Summary of account details (email, phone, address, pickup/delivery preference).
First order details (if placed) with date, time, and location/driver info.
Link to app download (if onboarding via website) or "Manage Subscription" button.
Features:
Include a quick survey: "How easy was sign-up? 1-5 stars" for feedback.

Additional Onboarding Enhancements
Progress Bar: Show a visual tracker (e.g., "Step 1 of 8") to keep customers motivated.
